# SubCompliance Pro

AI-powered subcontractor compliance management for General Contractors.

## What It Does

- Verify contractor licenses across 50 US states
- Track COI expirations (GL, Workers Comp, Auto)
- AI-powered document analysis
- Automated email reminders to subcontractors
- Risk scoring for each subcontractor
- Generate compliance reports

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## Deploy to Vercel

1. Fork this repo
2. Go to vercel.com
3. Import your forked repo
4. Click Deploy

Done. Your app will be live in 60 seconds.

## Local Development

```bash
npm install
npm run dev
```
